# phase4-sample-app

This is a Railway-ready Node.js sample application.

## Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the application: `npm start`

## Deployment

This application is configured to listen on `process.env.PORT` or default to 3000 and bind to `0.0.0.0`, making it suitable for deployment on platforms like Railway.
